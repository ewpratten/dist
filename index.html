Evan's Software Repository

This server hosts builds of various projects I've worked on.

Fedora repository

To add my Fedora repository to your system, run this command:

sudo dnf config-manager --add-repo https://dist.ewpratten.com/linux/fedora/ewpratten.repo

Once added, you can check what packages are available on your platform via:

dnf repoquery --repo ewpratten -i --refresh

Enabling development builds

Development builds are available for some packages. To enable them, run:

sudo dnf config-manager --set-enabled ewpratten-dev
